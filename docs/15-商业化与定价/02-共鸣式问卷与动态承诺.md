# 新用户共鸣式问卷（题库/逻辑/动态承诺）

更新时间：2026-01-30  
目的：用“自我陈述”引发共鸣，把用户从“我试试”推进到“我需要它”。

---

## 1. 问卷结构（最短 6 题，MVP 推荐 6–8 题）
- Q1 目标：Cut / Maintain / Bulk（必须）
- Q2 痛点主因：体重不动 / 夜宵 / 断练 / 睡不稳（必须）
- Q3 想先改善：食欲 / 训练 / 睡眠恢复 / 时间安排（必须）
- Q4 睡眠现状：规律 / 不稳定 / 经常熬夜 / 不确定（必须）
- Q5 记录偏好：拍照 / 条码搜索 / 手动（必须）
- Q6 14 天愿望：食欲稳定 / 体重变化 / 执行稳定 / 精力更好（推荐）
- Q7 可选：目前活动量（低/中/高）
- Q8 可选：是否有训练习惯（抗阻/瑜伽/无）

---

## 2. 题库（MVP 文案）
### Q1：你现在最在意哪件事？
- 我想减脂（Cut）
- 我想保持体重（Maintain）
- 我想增肌（Lean bulk）

### Q2：你最常遇到的情况是？
- 明明少吃了，体重还是不动
- 一忙就乱吃/夜宵
- 运动断断续续，坚持不下去
- 睡眠不稳，第二天没状态

### Q3：你最想先改善哪一项？
- 食欲控制（尤其晚上）
- 训练执行（该练的时候练）
- 睡眠恢复（睡够但不恢复）
- 时间安排（白天能量窗口混乱）

### Q4：最近 2 周你睡得怎么样？
- 很规律（差不多时间睡醒）
- 还行但不稳定
- 经常熬夜/早醒
- 不确定

### Q5：你更倾向哪种记录方式？
- 拍照就行（最省事）
- 条码/搜索（更准确）
- 我愿意手动录入（最可控）

### Q6：如果 14 天后你能明显更好，你希望是哪一种？
- 食欲更稳定（不再晚间崩）
- 体重开始向目标移动
- 训练执行更稳定
- 睡醒更有精神

## 3. 动态承诺（Paywall 前生成 1 句话）
规则（示例）：
- goal=cut 且 Q2=夜宵 → “我们会用睡眠恢复 + 进食截止窗，让晚间食欲更稳定。”
- goal=bulk 且 Q3=训练执行 → “我们会在恢复足够的日子推训练与热量盈余，避免‘练了却不长’。”
- Q4=经常熬夜 → “先把节律拉回正轨，再做热量目标，减脂更稳。”

输出字段：
- `personal_promise_key`
- `personal_promise_params`（可选）
