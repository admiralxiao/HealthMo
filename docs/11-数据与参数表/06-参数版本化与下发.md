# 06-参数版本化与下发

更新时间：2026-01-30

## 版本字段
- `params_version`: 例如 `2026.01.30`
- `bootstrap_version`: 例如 `2026.01.30`

## 下发
- `GET /config/bootstrap` 返回：flags + params + providers + versions
- 客户端缓存：TTL=1h（默认）
- 强制刷新：
  - Paywall 展示前
  - 购买/恢复购买后
  - 命中 kill switch 时

## 兼容策略
- 客户端需对未知字段“忽略但保留”；
- 关键字段缺失时用本地安全默认值（文档中定义）。
