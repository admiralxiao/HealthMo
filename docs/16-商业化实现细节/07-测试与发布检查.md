# 测试清单与发布检查（商业化）

更新时间：2026-01-30

## 1) 购买/恢复（必须）
- Weekly 购买成功→entitlement active→UI 解锁
- Yearly 购买成功→entitlement active
- userCancelled 不弹错误
- 弱网/断网可重试
- Restore 成功→entitlement active

## 2) 退款/过期回收（必须）
- 收到 server notifications 后 entitlement 回收
- 客户端刷新后降级
- premium API 403 → entitlement refresh → 正确提示

## 3) Offer 分层
- 默认 A 展示正确
- 停留 >=8s 弹 B（若允许）
- 7d paywall_view>=2 且 risk=low 触发 C（若允许）
- risk>=medium：B/C 不出现

## 4) kill switch
- kill_switch.payments=true：隐藏购买入口，仅 restore
- feature.paywall.enabled=false：全局关闭商业化

## 5) App Store 合规
- 自动续订/取消/restore/条款隐私 文案齐全
- 不做医疗承诺
- 价格与 IAP SKU 一致
