# Remote Config 字段（商业化）与默认值

更新时间：2026-01-30  
来源：GET /config/bootstrap（docs/12）

## 1) flags
- feature.paywall.enabled = true
- feature.paywall.offer_B = true
- feature.paywall.offer_C = false（默认关，灰度开）
- kill_switch.payments = false

## 2) params
- paywall.offer_trigger.seconds = 8
- paywall.offer_C.min_paywall_views_7d = 2
- paywall.entitlement.cache_ttl_sec = 86400（24h）
- risk.degrade.disable_discount_level = "medium"

## 3) providers
- paywall.offer_default = "A"
- paywall.layout_variant = "oura_v1"

## 4) per-user override（v1.1 预留）
bootstrap 可带 overrides（先留接口）：
```json
{
  "overrides": {
    "user:123": { "paywall.offer_default": "B" }
  }
}
```
