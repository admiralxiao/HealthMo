# 03-MVP范围与里程碑（2 周）

更新时间：2026-01-30

## MVP 范围（In/Out）
**In**
- Today 三卡：Energy/Recovery/Windows + Action Panel
- Food：拍照结构化 + 条码营养库匹配
- 活动识别：Core Motion 分段 + 步数强度分级 + MET 消耗
- 睡眠：只接入分期 → 恢复/LBI/窗口
- 商业化：问卷→预览→背书→Paywall（StoreKit2）
- 远程配置/风控底座（最小可用）
- 上线准备（订阅、隐私、监控回滚）

**Out（MVP 不做）**
- 医疗级诊断/治疗承诺
- 复杂训练计划库/社交/多硬件联动

## 两周里程碑
- D1–D3：数据模型、RemoteConfig、Entitlement、设计系统组件
- D4–D7：Today Energy + Food 最小闭环
- D8–D10：Sleep/Recovery/LBI + Windows + Action Panel
- D11–D12：Paywall/Offer 分层 + 风控接入
- D13：联调/测试/修复 + 审核物料
- D14：发布准备 + 监控回滚演练
