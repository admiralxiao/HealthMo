# 01-项目概览（Health App MVP）

更新时间：2026-01-30

适用：iOS 原生（Swift, iOS 15.6+）+ FastAPI + Postgres  
开发方式：Claude CLI 协作（仓库根目录 `claude.md` 为全局规范）

## 目标与定位
- 面向欧美用户的“饮食 + 运动 + 睡眠”健康管理 App（Oura 风：高级、克制、可信）
- MVP 目标：两周内跑通 **Today 闭环 + 商业化闭环 + 安全/风控底座**

## MVP 核心能力
1) 自动活动识别（Core Motion）：睡眠/静止/步行/跑步/骑行/乘驾  
2) 能量计算：全天总消耗、运动消耗；抗阻/瑜伽手动记录  
3) 饮食摄入：拍照结构化识别 + 条码营养库匹配  
4) 恢复指标：睡眠分期接入 → 体力/脑力恢复分 + LBI（瘦素平衡支持度）  
5) 节律窗口：光照/高专注/训练/进食截止/放松窗 → 行动面板  
6) 商业化：问卷→价值预览→技术背书→Paywall（周订主推+年卡承接+折扣分层）  
7) 安全/风控：服务端 entitlement 权威、折扣防薅、成本限频
