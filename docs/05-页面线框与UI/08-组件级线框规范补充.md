# 08-组件级线框规范补充

更新时间：2026-01-30

## MetricCard
- 圆角：22
- 内边距：16
- 间距：12
- 标题：13–14pt，字重 medium
- 数值：28–32pt，字重 semibold
- 说明：12–13pt，最多两行
- 可点击区域：整卡

## Action Panel
- 3 条以内；每条：动词标题 + 1 行原因 + CTA
- CTA 文案统一动词开头（Log / Start / Adjust）

## Explain Sheet
- 结构：What / Why / Next
- 最大 3 屏；用 bullet，避免长段落

## PlanPickerCard
- 整卡可点；选中态边框高亮
- 主推周订：$9.99/week；年卡：$59.99/year（显示折算）
