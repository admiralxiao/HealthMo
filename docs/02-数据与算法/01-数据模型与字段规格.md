# 01-数据模型与字段规格（API/存储一致）

更新时间：2026-01-30

## 统一约定
- 时间：ISO8601(UTC)；展示用 client_timezone
- 分数：0–100；比例：0–1

## 枚举
- activity_type: sleep|still|walk|run|bike|vehicle|strength|yoga
- intensity: light|moderate|vigorous
- sleep_stage: awake|light|deep|rem

## 核心对象（摘要）
### ActivitySegment
start_at,end_at,activity_type,steps?,distance_m?,intensity?,met,kcal

### FoodLog
source(photo|barcode|manual), items[], total_kcal, macros(P/F/C)

### SleepStage
start_at,end_at,stage

### Daily summaries
tdee_kcal, active_kcal, intake_kcal, net_kcal
physical_score, mental_score, lbi_score, sleep_debt_minutes
