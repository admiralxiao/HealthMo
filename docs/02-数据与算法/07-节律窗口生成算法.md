# 07-节律窗口生成算法（Windows）

更新时间：2026-01-30

```mermaid
flowchart TD
  A[昨日睡眠/节律] --> B[计算sleep_midpoint/chronotype]
  B --> C[恢复分]
  C --> D[生成窗口模板]
  D --> E[按目标与恢复修正\n训练强度/进食截止]
  E --> F[输出今日窗口 + 行动建议]
```

## 输出窗口（示例）
- light_window: wake+0~2h
- focus_window: wake+2~5h
- dip_window: wake+6~8h
- training_window: 按 recovery 调整
- eating_cutoff: bedtime-3h
- winddown: bedtime-90min~bedtime
